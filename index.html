Session: Getting Started

Objective Learn "everything you'll need to know about getting started with the Java programming language."

  1. Create a new HelloWorldApp class in IntelliJ.
    1. Right click on the src folder in the project window and select New -> Java Class
    2. Name the class HelloWorldApp
  2. Copy and paste the following code into the HelloWorldApp class. 
    /**
 * The HelloWorldApp class implements an application that
 * simply prints "Hello World!" to standard output.
 */
class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); // Display the string.
    }
}

Session: Equality, Relational, and Conditional Operators

Objective: Compare variables

Prepare: Read

  • The Equality, Relational, and Conditional Operators section in the tutorial

Assignment: Do the following:

  • Create a new project in IntelliJ with a Main class with a main method
  • Assign values to varibles, compare the variables in an if statement and print the result in the body of the if statement.
  • Make sure to do at least one of each >, >=, <, <=, ==, !=, &&, ||. Optionally you can try instanceof.

Here is an example: 

int x = 5;
int y = 6;
if (x < y) {
  System.out.println("x is 5 and is less than y which is 6");
}

if (x != y) {
  System.out.println("x is 5 and does not equal y which is 6");
}

int z = 7;
if (z >= 7) {
  System.out.println("z is 7 which is greater or equal to 7");
}

Classroom: Live coding and go over the nuances of the short circuit AND and OR operators.

Processing command line arguments

Objective: Loop through the command line arguments

Lessons: Ever noticed that the main signature has an array of arguments?

public static void main(String[] args)

In intelliJ you can pass arguments to a program. See my youtube CommandLineArgsIntelliJ video.

Obviously the arguments are Strings.

Assignment 1:

  • Create a new project in IntelliJ with a Main class with a main method
  • Run the main method and then edit the run configuration to add arguments
  • In a for loop print out the arguments
  • If an argument is "exit" then break out of the for loop which will end the program because there is no code after the for loop.
  • If an argument is "skip" then skip the rest of the code in the loop and continue to the top of the loop.
  • If the argument is "echo" then print "echo" twice.

Lesson continued

In Java you can change a String to an double by doing the following: 

String numberAsText = "3";
double number = Double.parseDouble(numberAsText);

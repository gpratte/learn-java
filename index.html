Session: Getting Started

Objective Learn "everything you'll need to know about getting started with the Java programming language."

  1. Create a new HelloWorldApp class in IntelliJ.
    1. Right click on the src folder in the project window and select New -> Java Class
    2. Name the class HelloWorldApp
  2. Copy and paste the following code into the HelloWorldApp class. 
    /**
 * The HelloWorldApp class implements an application that
 * simply prints "Hello World!" to standard output.
 */
class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); // Display the string.
    }
}

Session: Variables - Arrays

Objective: Create, add data and access data in arrays

Prepare: Read

  • The Arrays section in the tutorial

Assignment: Do the following:

  • Create an array of Strings for bad baby names of length 5
  • Create another array of Strings of length 10
  • Assign the first array to the second array twice

For example the array of bad baby names could be
"Hellzel", "Ahmiracle", "Baby Girl", "Elizabreth", "Kaizyle"

The second array will be
"Hellzel", "Ahmiracle", "Baby Girl", "Elizabreth", "Kaizyle", "Hellzel", "Ahmiracle", "Baby Girl", "Elizabreth", "Kaizyle"

Question: What happens when you try to access a cell of the array that is beyond the length of the array (e.g. the array is length 10 and you try to access the 11th cell)

Classroom: Look at, and code using the helper methods (e.g. search, sort), the Arrays class.

For the motivated: Read the Summary of Variables and do the Questions and Exercises: Variables
